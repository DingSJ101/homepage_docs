## 有向图

DFS形成了一颗树，定义边类型`(u,v)`

1. Tree Edge DFS树上的边
2. Forward Edge v是u的非儿子后继
3. Back Edge  与1.2.相反
4. Cross Edge 两个节点不构成前驱后继，共享祖先
