## Mysql使用

## 安装(Windows)

### 下载

1、网址  dev.mysql.com/downloads/mysql

2、Zip Archive 版本

3、不注册，直接下载解压

### 安装

1、管理员权限 进入 文件夹/bin

2、初始化 

```shell
mysqld --initialize 
```

3、启动服务

```powershell
net start mysql
>>>MySQL 服务正在启动 .....  
>>>MySQL 服务已经启动成功。 
```

### 使用

#### 登录

```shell
mysql -uroot -p
```
