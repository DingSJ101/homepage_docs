## frp内网穿透

### 服务器端

1. 下载frp [（link）](https://github.com/fatedier/frp/releases)，并解压

2. 修改`frps.ini`

3. ```
   # frps.ini
   [common]
   bind_port = 7000
   ```

4. 开启服务`./frps -c ./frps.ini`

### 客户端

1.2 同上

3. ```
   # frpc.ini
   [common]
   server_addr = x.x.x.x
   server_port = 7000
   
   [ssh]
   type = tcp
   local_ip = 127.0.0.1
   local_port = 22
   remote_port = 6000
   ```

4. `./frpc -c ./frpc.ini`
